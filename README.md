# SecureGuard: Risk-Based MPIN Security

## About
SecureGuard is a risk-based authentication system designed for MPIN reset flows. It evaluates real-time behavioral and environmental signals to dynamically adjust verification steps, preventing fraud while ensuring a seamless experience for legitimate users.

## Tech Stack
- Frontend: HTML/CSS (Prototype), React.js (Target)
- Backend: Node.js (Express)
- Database: MongoDB
- Presentation: LaTeX (Beamer)

## Prototype Note
The current UI and code included in this repository are prototypes built for the initial submission. We will be building out the full-scale application and risk engine logic during the hackathon event.
